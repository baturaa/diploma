
Batura Store

Простой адаптивный сайт-каталог кроссовок и одежды.

Структура проекта

- `landing page.html` — главная страница
- `landing page.css` — стили
- `images/` — изображения товаров и логотипов
- Остальные HTML-файлы — страницы с деталями товаров

Возможности

- Адаптивная верстка 
- Навигация между разделами
- Визуальные карточки товаров
- Используется Google Font: [Poppins](https://fonts.google.com/specimen/Poppins)

Заметки

- Все изображения локальные (в папке /images)
- Код структурирован и снабжен комментариями
- Атрибут `alt` добавлен ко всем изображениям
